yearning-sql审计平台安装

发表于 2018-02-28 | 分类于 运维分享

环境准备

安装依赖

安装python3.6

编辑安装文件 /usr/local/python3/Modules/Setup

安装mysql

安装nginx

访问 10.1.3.111 出现下面截图说明正常

安装yearning

修改配置文件 vim.cnf

启动inception

如果这个不会作者在install里面已经有编译好的只需要修改配置文件就可以了

测试连接inception

mysql -uroot -h127.0.0.1 -P6669

配置 yearning

[root@sql src]# vim deploy.conf

安装yearning 依赖包

注意事项

vim /usr/local/python3/lib/python3.6/site-packages/pymysql/connections.py //1108行左右

[root@sql src]# vim /usr/local/python3/lib/python3.6/site-packages/pymysql/cursors.pyt

如果怕修改出错 。可以去安装目录install 下直接吧里面的文件替换到python目录下

添加用户初始化

把编译好的文件复制到nginx html 目录下

重启nginx

启动django

访问deploy.conf 配置文件中ipaddress 填写的地址 默认账号: admin 密码:Yearning_admin

解决办法yum install zlib zlib-devel -y

问题2

解决办法

编辑安装文件 /usr/local/python3/Modules/Setup

然后从新编辑安装
